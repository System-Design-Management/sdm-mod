# Phase 3

## 目的層：何の感情を出したい？

純粋な恐怖

## 状態層：物理・心理的にプレイヤーの何を変える？

集中力を高めさせる。過剰に反応してしまう状態を引き出す。

## 変数層：そのために何の変数を操作する？

静かな感じ。急に仕掛ける。

## 評価層：どんな反応が出るか確認

びっくりさせる。

### シーン３：図書館3階

部屋に入る。

ゾンビと戦う。ユーザーの座標に応じてスポーンさせることで、意図しない挙動を防ぐ。視界から消えたらデスポーンさせるなど、撒ける手段も設ける（倒す前提にしない）。

ゾンビはどのルートでも必ず2~3体は鉢合わせる構成にしたい。変なルートで行くと遭遇する数が増えるのでも問題ない。
