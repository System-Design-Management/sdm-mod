# Phase 2. 緊張

図書館前にテレポートさせ、ゲームを開始させる。この Phase ではコンセプトを理解させるためのチュートリアル程度に済ませる。

## 目的層：何の感情を出したい？

緊張

## 状態層：物理・心理的にプレイヤーの何を変える？

若干心拍数が上がるぐらい

## 変数層：そのために何の変数を操作する？

敵数は少なめ。演出も弱め。

## 評価層：どんな反応が出るか確認

恐れさせすぎず、退屈させない状態。

### シーン１：図書館入口前

プレイヤーは責任を負うため、鉄パイプ（木の棒でも可）で図書館に飛び込もうとする。プレイヤーは、研究室内の人物とトランシーバーで連絡をとっているものとする。

前方に警官（その他銃を公的に使用できる職業でも可）が倒れているのを確認。

その警官が持っている銃を手に取る。

図書館入口前にいるゾンビを例に、銃の使い方のチュートリアル。とはいえ、できるだけ自然な流れで。

銃の使い方を教えるとともに、銃を使わなくてもいい旨を伝えておく。

### シーン２：図書館に入館

中に入る。学生証を用いてゲート内へ。3階を目指す。

2階：鉄格子の向こうにゾンビを配置

（上からゾンビが落ちてくる）

検索用 PC は入口付近に配置する想定。PC を使っている途中でゾンビが襲ってこないようにする。

この Phase では攻撃してくるゾンビはいないが、ゾンビの恐怖は植え付けるようにする。イメージとしてはお化け屋敷の恐怖に近い。
