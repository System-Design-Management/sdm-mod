# Phase 5

この Phase では、流れを途切らせないために、死亡時のリスポーン処理を行わない。逃げ切れたらハッピーエンド、死亡すればバッドエンド。

## 目的層：何の感情を出したい？

焦り・恐怖

## 状態層：物理・心理的にプレイヤーの何を変える？

考える余裕をなくさせる

## 変数層：そのために何の変数を操作する？

敵数を増やす。音量を上げてもいいかもしれない。

## 評価層：どんな反応が出るか確認

慌てふためかせたい。

### シーン５：ラストのダッシュ

大量にゾンビが襲いかかってくる。

銃では対処できず逃げるしかないということを説明（プレイヤーの前ではなく後ろにゾンビがいる前提）。

とにかく走らせる。出口前のゲート（学生証必須）でドタバタさせるのもあり。操作がややこしくならない程度に。

逃げ切れればハッピーエンド、捕まればバッドエンド。
